Unified Design Reports

Export comprehensive Markdown or JSON reports containing layout specs, typography rules, color accessibility, and technology stack info.

Stack Detection

Identify the frameworks, analytics, CDNs, and CMS platforms running the site through an intelligent stack analysis engine.

Core Web Vitals

Monitor Real User Metrics (RUM) including Largest Contentful Paint (LCP) and Cumulative Layout Shift (CLS) on the fly.

How to Install

1. Clone the Repository

Download or clone the source code from GitHub directly to your local machine to get started.

2. Build the Extension

Run the build commands using Bun or npm to compile the TypeScript, Tailwind, and React code into the dist/ folder.

3. Load into Browser

Navigate to chrome://extensions, enable Developer Mode, and select "Load unpacked" targeting the compiled dist/ folder.

How to Use

1. Open the Extension

Click the Kaes Keid icon in your toolbar. It starts on the Stack tab, immediately analyzing the page's technology fingerprint.

2. Select an Analyzer Tab

Switch to any other tab (Overview, Colors, Type, Assets). The CSS Inspector activates automatically, enabling hover tooltips and element selection.

3. Export or Go Back

Extract colors, sizes, and layout data with the export functionality. Switch back to the Stack tab anytime to stop the Inspector cleanly.
