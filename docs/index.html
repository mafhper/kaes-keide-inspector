Deep Property Inspection

No more digging through DevTools. Instantly view box model, borders, shadows, and typography properties completely parsed and visualized.

Grid & Flexbox Analyzer

Advanced analysis for modern CSS layouts. See grid template areas, track sizes, flex-grow rules, and alignment settings at a glance.

Color Palette Extraction

Automatically extract the entire color scheme used on the page. Download the palette as CSS variables, SCSS, or Design Tokens instantly.

Unified Design Reports

Export comprehensive Markdown or JSON reports containing layout specs, typography rules, color accessibility, and technology stack info.

Stack Detection

Identify the frameworks, analytics, CDNs, and CMS platforms running the site through an intelligent stack analysis engine.

Core Web Vitals

Monitor Real User Metrics (RUM) including Largest Contentful Paint (LCP) and Cumulative Layout Shift (CLS) on the fly.
